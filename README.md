# BenBen Better

在任意洛谷页面快速发送犇犇。

## 安装

### 安装 Tampermonkey

在浏览器扩展商店搜索并安装 Tampermonkey：

- Chrome / Edge：访问 Chrome 网上应用店
- Firefox：访问 Firefox 附加组件商店
- Safari：在 App Store 搜索

### 安装脚本

1. 点击 Tampermonkey 图标，选择「管理面板」
2. 点击左侧「+」号新建脚本
3. 复制 `benben-better.user.js` 的全部代码并粘贴
4. 保存（Ctrl+S）

### 开始使用

访问洛谷任意页面，右下角会出现一个紫色浮动按钮。

## 使用方法

- 点击右下角按钮打开发送窗口
- 输入内容后点击「发送」或按 Ctrl+Enter
- 点击窗口外部或关闭按钮可关闭窗口

## 常见问题

**按钮不显示**

检查 Tampermonkey 是否已启用，刷新页面重试。

**发送失败**

确认已登录洛谷账号，检查网络连接。

**动画卡顿**

尝试清理浏览器缓存或更新浏览器。
